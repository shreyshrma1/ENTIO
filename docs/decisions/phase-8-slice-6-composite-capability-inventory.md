# Phase 8 Slice 6 Composite Capability Inventory

This inventory is the allowlist for composite preparation. Every approved composite delegates reads to existing bounded project/entity/FIBO services and emits only existing `WebStageChangeRequest` typed operations through `AiTypedEditCapabilityAdapter`. No entry may emit raw RDF, mutate shared staging, approve, apply, or write ontology sources.

| Capability | User intent | Existing reads | Typed outputs | Maximum output | Ambiguity conditions | Unsupported conditions | Required validation | Parity tests | Status |
| --- | --- | --- | --- | ---: | --- | --- | --- | --- | --- |
| `prepare_class_model` | Create or enrich a bounded class model | project map, entity detail, hierarchy, search | approved class, label, definition, hierarchy, domain/range requests | 20 operations | colliding label/IRI, unresolved parent/property | complex OWL expression or raw RDF required | ordinary typed preparation plus batch validation | compare every prepared request with manual `AiTypedEditCapabilityAdapter.prepare` | `APPROVED` |
| `prepare_property_model` | Create or enrich bounded object/datatype properties | entity detail, usage, domain/range neighborhood | approved property, label, definition, domain, range requests | 20 operations | property kind, domain, or range unresolved | complex property axiom required | ordinary typed preparation plus batch validation | compare every prepared request with manual preparation | `APPROVED` |
| `prepare_domain_model_batch` | Prepare a bounded multi-entity domain model package | project map, search, neighborhoods | approved class/property/label/definition/hierarchy requests | 20 operations per batch | identity or source ownership unresolved | request needs more than 100 task edits or unsupported axioms | ordinary typed preparation, dependency, task, and batch limits | deterministic 50-edit multi-batch parity test | `APPROVED` |
| `prepare_external_reuse` | Reuse a selected pinned FIBO element | pinned FIBO search/detail/dependency review | existing external-reuse typed request | 20 operations | candidate or dependency selection unresolved | unpinned/unapproved external ontology | FIBO dependency review and ordinary typed preparation | compare with manual FIBO proposal preparation | `CLARIFICATION_REQUIRED` |
| `prepare_entity_refactor` | Rename or restructure existing entities | entity detail, usage, hierarchy, deletion dependency review | approved label/hierarchy/delete requests | 20 operations | target identity or dependent treatment unresolved | merge/split semantics or arbitrary graph rewrite required | dependency review and ordinary typed preparation | compare with manual label/hierarchy/deletion preparation | `CLARIFICATION_REQUIRED` |

Only the three `APPROVED` entries are exposed by the Slice 6 orchestrator. The other candidates return structured clarification results and cannot silently degrade to raw graph operations.
